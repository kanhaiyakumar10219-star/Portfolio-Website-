# Kanhaiya Kumar Portfolio

Hi 👋  
This is my personal portfolio website created using HTML, CSS, and JavaScript.  
I made this project to improve my frontend development skills and showcase my projects, skills, and contact information.

The website is fully responsive and works on both desktop and mobile devices.


## 🚀 Features

- Modern dark theme UI
- Responsive design
- Smooth animations
- Skill progress bars
- Project popup modal
- Multiple project screenshots
- Fullscreen image preview
- Contact form using EmailJS
- Social media links
- Resume download button


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- EmailJS


## 📂 Projects Included

- Student Complaint Management System
- Weather App
- Smart Attendance Tracker System


## ▶️ How to Run

1. Download or clone this repository
```bash
git clone https://github.com/kanhaiyakumar10219-star/portfolio.git
```

2. Open the project folder
3. Run `index.html` in your browser
